## Featured projects

### RoleStride

AI-enabled job-screening and opportunity-assessment platform.

RoleStride is my flagship AI decision-engine project. It uses structured vacancy research, candidate-evidence governance, OpenAI API-based assessment, scoring policy, decision thresholds, benchmark calibration and telemetry to support repeatable career decision-making.

My role: product owner, solution architect and AI workflow designer. I defined the product vision, scoring policy, source hierarchy, prompt governance, test strategy, benchmark cases and acceptance criteria, and used AI-assisted engineering to implement and iterate the product.

### Mock Exam Engine

Local-first assessment and question-bank platform.

Mock Exam Engine demonstrates productisation, assessment design, local-first architecture, user workflow design, content validation, regression testing and learning-product design. It evolved from a personal AgilePgM revision tool into a reusable mock exam platform with importable question banks, configurable practice sessions, mock exams, explanations and progress tracking.
