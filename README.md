## Featured projects

### RoleStride

AI-enabled job-screening and opportunity-assessment platform.

RoleStride is my flagship AI-enabled decision-system project. It was designed to bring structure, evidence and governance to senior opportunity assessment, using structured vacancy research, candidate-evidence governance, OpenAI API-based assessment, scoring policy, decision thresholds, route logic, benchmark calibration and telemetry.

My role: product owner, solution architect and AI workflow designer. I defined the product vision, user workflow, source hierarchy, scoring policy, prompt governance, evidence guardrails, benchmark cases, acceptance criteria and delivery controls, using AI-assisted engineering to implement and iterate the product.

### Mock Exam Engine

Local-first assessment and question-bank platform.

Mock Exam Engine demonstrates productisation, assessment design, local-first architecture, user workflow design, content validation, regression testing and learning-product design. It evolved from a personal AgilePgM revision tool into a reusable mock exam platform with importable question banks, configurable practice sessions, mock exams, explanations and progress tracking.

Future roadmap areas include AI-assisted training-material processing, question generation and revision-support workflows.
